# Financial Data Analysis Pipeline

## Project Overview

Automated ETL pipeline for financial market data analysis. Extracts stock data from Yahoo Finance API, processes and stores in MongoDB, and visualizes insights through Power BI dashboards.

**Author:** Hasara Wijayarathna  
**Tech Stack:** Python, Yahoo Finance API, MongoDB, AWS S3, Power BI  
**Data Source:** Real-time stock market data via Yahoo Finance

## Business Objectives

- Build automated data pipeline for financial data collection
- Store and manage time-series financial data efficiently
- Analyze stock price trends and volatility
- Create interactive dashboards for investment insights
- Demonstrate ETL and cloud integration skills

## Tech Stack

**Data Extraction:** Yahoo Finance API, Python (yfinance)  
**Data Processing:** pandas, NumPy  
**Data Storage:** MongoDB (NoSQL database)  
**Cloud Storage:** AWS S3  
**Visualization:** Power BI, matplotlib, seaborn  
**Automation:** Python scripts
